## О проекте

Программная реализация алгоритмов поиска линеаризующих множеств (LP-множеств) мощности k с наибольшей нижней границей вероятности линеаризации системы булевых функций.

## Описание модулей

1. **`system_of_Boolean_function.py`** - класс система булевых функций + алгоритмы поиска линеаризующих множеств
   - **brute_force_linearization_check** - алгортим полного перебора
   - **calculate_linearization_probability** - алгоритм поиска LP-множеств мощности k с наибольшей нижней границей (алгоритм 1)
   - **calculate_linearization_probability_optimized** - оптимизация алгоритма поиска LP-множеств мощности k с наибольшей нижней границей (алгоритм 2)
   - **calculate_refined_probability** - алгоритм поиска LP-множеств мощности k с наибольшей нижней границей с уточнениями

2. **`experiment.py`** - эксперименты, описанные в работе
   - **run_speed_experiment** - сравнение времени работы алгоритма полного перебора, алгоритма 1, алгоритма 1 с уточнениями при изменении мощности рассматриваемых множеств k
   - **run_scalability_experiment** - сравнение времени работы алгоритма полного перебора, алгоритма 1, алгоритма 1 с уточнениями при изменении числа переменных в системе n
   - **run_accuracy_experiment** - сравнение точности работы работы алгоритмов 1 и 1 с уточнениями при изменении мощности рассматриваемых множеств k
   - **run_optimization_comparison_experiment** - сравнение времени работы алгоритма 1, алгоритма 2 при изменении мощности рассматриваемых множеств k
   - **run_random_system_analysis** - изменение мощности множеств EX и величины EX/ALL при изменении мощности рассматриваемых множеств k
